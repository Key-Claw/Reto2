# Reto2 – Programación (1º DAW)

Trabajo realizado por Raúl y Cristian.
Este repositorio recoge nuestras soluciones a los ejercicios del Reto 2, que abarcan desde el apartado A hasta el L. Cada ejercicio plantea un problema distinto que requiere aplicar lógica, estructuras de control y gestión de datos en Java.

Los retos incluyen situaciones como ordenar apuntes, simular tareas en paralelo, validar etapas de una vuelta ciclista, encontrar caminos en laberintos, comprobar años cúbicos, calcular días para coincidir con amigos y otros problemas similares. Cada apartado se aborda de forma independiente y con su propia implementación.

El desarrollo se ha organizado con Git y GitHub, trabajando en ramas y fusionándolas en la rama principal una vez completadas.
